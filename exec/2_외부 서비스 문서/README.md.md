## 외부 서비스 문서

### 🎭 Frontend

- Vue & Vuex

  ```vue
  npm install -g @vue/cli 
  ```
  
- Axios

  ``` vue
  npm install axios
  ```

- TailwindCSS

  ``` vue
  npm install -D tailwindcss@latest postcss@latest autoprefixer@latest
  ```

  

### 🩺 Backend

- JPA
- jwt, security, oauth



### 🧨Unity

- Unity
- Photon



### 🎁 Build & Deploy

- Frontend bulid process

  ``` vue
  cd ../frontend
  npm install
  npm run build
  npm run serve
  ```

- Backend build process

  ``` java
  cd backend;
  ./gradlew bulid
  ```
